## Prefix Key

The prefix key has been remapped from the default `C-b` to `C-Space` for easier access:

```bash tmux.conf
unbind C-b
set -g prefix C-Space
bind C-Space send-prefix
```

<Note>
**Prefix Key**: `Ctrl+Space`

All Tmux commands start with the prefix key. Press `Ctrl+Space`, release, then press the command key.
</Note>

## Keybinding Categories

### Copy Mode

Enter copy mode to navigate, search, and copy text from your terminal history.

#### Entering Copy Mode

| Keybinding | Action |
|------------|--------|
| `Prefix + [` | Enter copy mode |

```bash tmux.conf
bind-key [ copy-mode
```

<Tip>
Once in copy mode, you can use Vim keybindings to navigate (h, j, k, l, w, b, etc.) because `mode-keys vi` is enabled.
</Tip>

#### Vi-Style Copy Mode Bindings

Once in copy mode, use these Vim-inspired keybindings:

| Keybinding | Action | Command |
|------------|--------|----------|
| `v` | Begin selection | `begin-selection` |
| `y` | Copy selection (yank) | `copy-selection` |
| `V` | Toggle rectangle selection | `rectangle-toggle` |

```bash tmux.conf
bind-key -T copy-mode-vi 'v' send -X begin-selection
bind-key -T copy-mode-vi 'y' send -X copy-selection
bind-key -T copy-mode-vi 'V' send -X rectangle-toggle
```

#### Copy Mode Workflow

1. **Enter copy mode**: `Prefix + [`
2. **Navigate**: Use Vim motions (h/j/k/l, w/b, f/F, etc.)
3. **Start selection**: Press `v`
4. **Extend selection**: Use Vim motions to select text
5. **Copy**: Press `y` to yank (copy) the selection
6. **Exit**: Copy mode exits automatically after copying

<Tip>
For rectangular (block) selection, press `V` instead of `v`. This is useful for selecting columns of text.
</Tip>

## Standard Tmux Commands

With the `C-Space` prefix, you can use all standard Tmux commands:

### Session Management

| Keybinding | Action |
|------------|--------|
| `Prefix + d` | Detach from session |
| `Prefix + $` | Rename session |
| `Prefix + s` | List and switch sessions |

### Window Management

| Keybinding | Action |
|------------|--------|
| `Prefix + c` | Create new window |
| `Prefix + ,` | Rename window |
| `Prefix + n` | Next window |
| `Prefix + p` | Previous window |
| `Prefix + 1-9` | Switch to window by number |
| `Prefix + &` | Kill window |

<Note>
Windows start at index 1 (not 0) in this configuration due to `set -g base-index 1`.
</Note>

### Pane Management

| Keybinding | Action |
|------------|--------|
| `Prefix + %` | Split vertically |
| `Prefix + "` | Split horizontally |
| `Prefix + o` | Cycle through panes |
| `Prefix + x` | Kill pane |
| `Prefix + z` | Toggle pane zoom |
| `Prefix + {` | Move pane left |
| `Prefix + }` | Move pane right |
| `Prefix + q` | Show pane numbers |

### Mouse Support

Mouse support is enabled (`set -g mouse on`), allowing:

- **Click** to select pane
- **Drag** pane borders to resize
- **Scroll** to navigate history (automatically enters copy mode)
- **Click** on window names in status bar to switch windows

## Quick Reference

### Most Common Commands

```bash
# Sessions
Prefix + d              Detach session
Prefix + s              List sessions

# Windows
Prefix + c              New window
Prefix + n/p            Next/Previous window
Prefix + 1-9            Go to window number

# Panes
Prefix + %              Split vertical
Prefix + "              Split horizontal
Prefix + o              Next pane
Prefix + z              Zoom pane

# Copy Mode
Prefix + [              Enter copy mode
v                       Begin selection (in copy mode)
y                       Copy selection (in copy mode)
V                       Rectangle selection (in copy mode)
```

<Tip>
To paste copied text, use `Prefix + ]` (standard Tmux paste command).
</Tip>
