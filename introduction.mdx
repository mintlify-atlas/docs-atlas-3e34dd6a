## Welcome to Your Modern Development Environment

This dotfiles repository provides a cohesive, keyboard-driven development environment for macOS. Built around the philosophy of efficiency and consistency, these configurations bring vi-mode navigation and unified aesthetics across your entire workflow.

## What Makes This Special

### Unified Vi-Mode Navigation
Consistent keyboard-driven navigation across all tools. Whether you're in Neovim, Tmux copy mode, or managing windows with AeroSpace, the same muscle memory applies.

### Beautiful Aesthetics
- **Catppuccin Theme**: Used in Tmux for a cohesive color palette
- **OneDark Colorscheme**: Primary Neovim theme with custom visual highlights
- **CoolNight Theme**: Alacritty terminal theme
- **0xProto Nerd Font Mono**: Professional monospace font with icon support

### Modern Development Setup
- **LazyVim**: Pre-configured Neovim distribution with sensible defaults
- **LSP Support**: Full language server protocol integration
- **Plugin Ecosystem**: Obsidian integration, LeetCode support, and more
- **GPU Acceleration**: Alacritty provides blazing-fast terminal rendering
- **True Color Support**: 256-color and RGB support across all terminals

### Productivity Features
- **Tiling Window Management**: AeroSpace brings i3-like window management to macOS
- **Tmux Sessions**: Persistent terminal sessions with easy workspace switching
- **Mouse Support**: Optional mouse integration in Tmux when needed
- **Custom Keybindings**: Optimized shortcuts that don't conflict with system defaults

## Philosophy

This configuration prioritizes:
1. **Keyboard Efficiency**: Minimal reliance on mouse interaction
2. **Consistency**: Similar keybindings and behavior across tools
3. **Performance**: GPU acceleration and optimized settings
4. **Extensibility**: Easy to customize and extend with your preferences
5. **Modern Features**: Latest tools and best practices

## Configuration Highlights

### Neovim (LazyVim)
- Multiple colorscheme options (Kanagawa, Gruvbox, Rose Pine, Tokyo Dark, Everforest)
- Custom visual selection highlighting
- Obsidian note-taking integration
- LeetCode plugin for coding practice
- Mason for LSP/formatter/linter management

### Alacritty
- 80% opacity with blur effects
- Buttonless window decorations
- 10px padding for aesthetics
- Maximized startup mode
- 16pt font size for readability

### Tmux
- Ctrl+Space prefix (instead of Ctrl+b)
- Vi-mode copy and paste
- Mouse support enabled
- Window renumbering
- Base index starts at 1
- 10ms escape time for responsiveness

### AeroSpace
- Alt-based keybindings (Alt+hjkl for navigation)
- Multiple workspace support (1-9 and A-Z)
- Auto orientation based on monitor dimensions
- Mouse follows focus
- Service mode for advanced operations
- Accordion and tiles layout modes
