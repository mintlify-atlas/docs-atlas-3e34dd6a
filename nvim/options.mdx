## Overview

This configuration keeps custom options minimal, relying primarily on LazyVim's sensible defaults. Custom options are defined in `lua/config/options.lua`.

## Custom Options

### Root Directory Specification

```lua
vim.g.root_spec = { "cwd" }
```

**Purpose**: Controls how LazyVim determines the "root directory" for projects.

**Default LazyVim Behavior**: Searches for common root markers like `.git`, `package.json`, `Makefile`, etc.

**Custom Behavior**: Forces the root directory to always be the current working directory (`cwd`) where Neovim was launched.

**Use Case**: This is useful when:
- Working in monorepos where you want to stay in a specific subdirectory
- You prefer explicit control over the working directory
- Avoid automatic directory changes based on file markers

### Visual Mode Highlighting

While not in `options.lua`, there's a custom highlight defined in `autocmds.lua`:

```lua
vim.api.nvim_set_hl(0, "Visual", { bg = "#ffff00", fg = "#000000" })
```

**Purpose**: Sets visual selection highlighting to bright yellow background with black foreground.

**Colors**:
- Background: `#ffff00` (bright yellow)
- Foreground: `#000000` (black)

**Effect**: Makes visual selections highly visible with maximum contrast.

### UI Enhancements

- **fillchars**: Custom characters for splits, folds, and diffs
- **listchars**: Visible whitespace characters
- **smoothscroll**: Smooth scrolling when available
- **statuscolumn**: Custom status column with line numbers and fold indicators

## Configuration File Location

The full options configuration file:

```
lua/config/options.lua
```

```lua
-- Options are automatically loaded before lazy.nvim startup
-- Default options that are always set: https://github.com/LazyVim/LazyVim/blob/main/lua/lazyvim/config/options.lua
-- Add any additional options here
-- vim.g.snacks_animate = false
vim.g.root_spec = { "cwd" }
```

## Adding Custom Options

To add more custom options, edit `lua/config/options.lua`:

```lua
-- Example: Set tab width to 4 spaces
vim.opt.tabstop = 4
vim.opt.shiftwidth = 4

-- Example: Enable line wrapping
vim.opt.wrap = true

-- Example: Set scroll offset to 8
vim.opt.scrolloff = 8
```

<Note>
  Custom options in this file will override LazyVim defaults. Add options here sparingly to maintain compatibility with LazyVim's opinionated setup.
</Note>

## Global Variables

Global variables (set with `vim.g.*`) are often used by plugins:

```lua
vim.g.root_spec = { "cwd" }           -- LazyVim root detection
-- vim.g.snacks_animate = false       -- Snacks.nvim animations
```
