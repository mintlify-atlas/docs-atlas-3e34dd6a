## Overview

Alacritty is a modern, GPU-accelerated terminal emulator focused on simplicity and performance. This configuration provides a clean, customizable setup with window transparency, blur effects, and access to 150+ community themes.

<Note>
Alacritty uses TOML format for configuration. The main config file is located at `~/.config/alacritty/alacritty.toml`
</Note>

## Window Configuration

The window settings create a clean, modern appearance with transparency and blur effects:

```toml alacritty/alacritty.toml
[window]
padding.x = 10
padding.y = 10

decorations = "Buttonless"
opacity = 0.8
blur = true
startup_mode = "Maximized"
```

### Window Options Explained

<Accordion title="Padding">
- **padding.x**: 10 pixels horizontal padding
- **padding.y**: 10 pixels vertical padding

Provides comfortable spacing between the window edge and terminal content.
</Accordion>

<Accordion title="Decorations">
Set to `"Buttonless"` to hide the window buttons while maintaining the title bar. Creates a minimal, distraction-free interface.
</Accordion>

<Accordion title="Opacity & Blur">
- **opacity**: Set to `0.8` (80% opaque) for a translucent effect
- **blur**: Enables background blur for a modern glass effect

This combination creates a sleek, semi-transparent terminal that lets you see content behind it while maintaining readability.
</Accordion>

<Accordion title="Startup Mode">
Set to `"Maximized"` to launch Alacritty in fullscreen mode automatically.
</Accordion>

## Font Configuration

The configuration uses the 0xProto Nerd Font, which includes programming ligatures and icon support:

```toml alacritty/alacritty.toml
[font]
normal.family = "0xProto Nerd Font Mono"
size = 16
```

<Tip>
**Nerd Fonts** include thousands of icons and glyphs that enhance terminal applications, file managers, and status lines. Make sure to install [0xProto Nerd Font](https://www.nerdfonts.com/) for full compatibility.
</Tip>

### Font Size

The default font size is set to **16pt**, providing excellent readability on modern high-DPI displays.

## Environment Variables

Alacritty sets specific environment variables for optimal terminal behavior:

```toml alacritty/alacritty.toml
[env]
TERM = "xterm-256color"
```

This ensures:
- Full 256-color support
- Proper terminal capabilities detection
- Compatibility with tmux and other terminal multiplexers

## Theme System

### Current Theme

The configuration imports themes using the `general.import` directive:

```toml alacritty/alacritty.toml
general.import = [
    "~/.config/alacritty/themes/themes/coolnight.toml"
]
```

Currently using: **coolnight** - a dark theme with cool blue tones.

## Switching Themes

To change your theme, simply update the import path in your `alacritty.toml`:

1. Open the configuration file:
   ```bash
   nvim ~/.config/alacritty/alacritty.toml
   ```

2. Change the theme import:
   ```toml
   general.import = [
       "~/.config/alacritty/themes/themes/YOUR_THEME.toml"
   ]
   ```

3. Save the file - changes apply immediately!

<Note>
Alacritty automatically reloads configuration changes. No restart required.
</Note>
