## Welcome to Desktop Configs

A carefully curated collection of configuration files for creating a productive, beautiful, and efficient development environment. Built on LazyVim, this setup combines powerful terminal tools with a modern tiling window manager to maximize your workflow.

<Note>
  This configuration is designed for macOS but can be adapted for Linux systems with minor modifications.
</Note>

## Key Features

<Columns cols={2}>
  <div>
    ### Editor
    
    - **LazyVim** as the base configuration
    - Multiple colorscheme options (OneDark, Kanagawa, Rose Pine, Gruvbox)
    - Language support for Python, Go, TypeScript, C/C++, and more
    - GitHub Copilot integration
    - Custom plugins for Obsidian, LeetCode, and more
  </div>
  
  <div>
    ### Terminal & Window Management
    
    - **Alacritty** with 100+ theme options
    - Custom blur and opacity effects
    - **Tmux** with Vim-style navigation
    - **AeroSpace** tiling window manager
    - Seamless workspace switching
  </div>
</Columns>

## Quick Start

Get up and running with your new development environment:

<Steps>
  <Step title="Install Prerequisites">
    Install Homebrew and required packages for your system.
  </Step>
  
  <Step title="Clone Configuration">
    Clone the config repository and symlink files to their proper locations.
  </Step>
  
  <Step title="Launch Tools">
    Open Neovim, Alacritty, and AeroSpace to complete the setup.
  </Step>
  
  <Step title="Customize">
    Adjust themes, keybindings, and plugins to match your preferences.
  </Step>
</Steps>

## Philosophy

This configuration prioritizes:

- **Simplicity**: Clean, minimal configuration that's easy to understand
- **Performance**: Fast startup times and responsive editing
- **Aesthetics**: Beautiful themes and consistent visual design
- **Productivity**: Optimized keybindings and workflow automation
- **Extensibility**: Easy to customize and extend with your own preferences

<Tip>
  All configuration files use modern formats: TOML for Alacritty and AeroSpace, Lua for Neovim.
</Tip>
