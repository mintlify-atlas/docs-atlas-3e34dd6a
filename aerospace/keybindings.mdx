## Main Mode Keybindings

The main binding mode is the default mode for everyday window management. All keybindings use the `alt` modifier as the primary key.

### Layout Switching

Switch between different window layout modes:

| Keybinding | Command | Description |
|------------|---------|-------------|
| `alt-slash` | `layout tiles horizontal vertical` | Toggle tiles layout between horizontal and vertical |
| `alt-comma` | `layout accordion horizontal vertical` | Toggle accordion layout between horizontal and vertical |

```toml
alt-slash = 'layout tiles horizontal vertical'
alt-comma = 'layout accordion horizontal vertical'
```

### Focus Navigation

Navigate between windows using Vim-style directional keys:

| Keybinding | Command | Description |
|------------|---------|-------------|
| `alt-h` | `focus left` | Focus window to the left |
| `alt-j` | `focus down` | Focus window below |
| `alt-k` | `focus up` | Focus window above |
| `alt-l` | `focus right` | Focus window to the right |

```toml
alt-h = 'focus left'
alt-j = 'focus down'
alt-k = 'focus up'
alt-l = 'focus right'
```

<Tip>
The h/j/k/l keys follow the Vim convention for navigation, making it intuitive for developers familiar with Vim keybindings.
</Tip>

### Window Movement

Move windows in different directions within the current workspace:

| Keybinding | Command | Description |
|------------|---------|-------------|
| `alt-shift-h` | `move left` | Move window to the left |
| `alt-shift-j` | `move down` | Move window down |
| `alt-shift-k` | `move up` | Move window up |
| `alt-shift-l` | `move right` | Move window to the right |

```toml
alt-shift-h = 'move left'
alt-shift-j = 'move down'
alt-shift-k = 'move up'
alt-shift-l = 'move right'
```

### Window Resizing

Resize the focused window:

| Keybinding | Command | Description |
|------------|---------|-------------|
| `alt-shift-minus` | `resize smart -50` | Decrease window size by 50 pixels |
| `alt-shift-equal` | `resize smart +50` | Increase window size by 50 pixels |

```toml
alt-shift-minus = 'resize smart -50'
alt-shift-equal = 'resize smart +50'
```

<Note>
The `resize smart` command automatically determines the best resize direction based on the window's position in the layout.
</Note>

## Service Mode Keybindings

Service mode provides advanced window management operations. Press `alt-shift-semicolon` to enter this mode.

### Configuration and Layout Management

| Keybinding | Command | Description |
|------------|---------|-------------|
| `esc` | `reload-config`, `mode main` | Reload configuration and return to main mode |
| `r` | `flatten-workspace-tree`, `mode main` | Reset layout by flattening workspace tree |
| `f` | `layout floating tiling`, `mode main` | Toggle between floating and tiling layout |
| `backspace` | `close-all-windows-but-current`, `mode main` | Close all windows except the focused one |

```toml
[mode.service.binding]
esc = ['reload-config', 'mode main']
r = ['flatten-workspace-tree', 'mode main'] # reset layout
f = ['layout floating tiling', 'mode main'] # Toggle between floating and tiling layout
backspace = ['close-all-windows-but-current', 'mode main']
```

<Warning>
The `backspace` command will close all windows except the currently focused one. Use with caution!
</Warning>

### Join Windows

Join the focused window with an adjacent window in a specific direction:

| Keybinding | Command | Description |
|------------|---------|-------------|
| `alt-shift-h` | `join-with left`, `mode main` | Join with window to the left |
| `alt-shift-j` | `join-with down`, `mode main` | Join with window below |
| `alt-shift-k` | `join-with up`, `mode main` | Join with window above |
| `alt-shift-l` | `join-with right`, `mode main` | Join with window to the right |

```toml
alt-shift-h = ['join-with left', 'mode main']
alt-shift-j = ['join-with down', 'mode main']
alt-shift-k = ['join-with up', 'mode main']
alt-shift-l = ['join-with right', 'mode main']
```

<Note>
Joining windows creates a container that groups them together, useful for organizing related windows within a workspace.
</Note>

## Customization

You can customize any keybinding by editing `~/.aerospace.toml`. The configuration supports:

- **Modifiers**: `cmd`, `alt`, `ctrl`, `shift`
- **Letters**: `a-z`
- **Numbers**: `0-9`, `keypad0-keypad9`
- **F-keys**: `f1-f20`
- **Special keys**: `minus`, `equal`, `period`, `comma`, `slash`, `backslash`, `quote`, `semicolon`, `backtick`, `leftSquareBracket`, `rightSquareBracket`, `space`, `enter`, `esc`, `backspace`, `tab`
- **Arrows**: `left`, `down`, `up`, `right`

For the complete list of available commands, visit the [official AeroSpace commands documentation](https://nikitabobko.github.io/AeroSpace/commands).
