## System Requirements

### Operating System

<Note>
  This configuration is designed specifically for macOS. While Neovim, Tmux, and Alacritty can run on Linux and other Unix-like systems, AeroSpace is macOS-only.
</Note>

- **macOS 10.15 (Catalina) or later** - Required for all components
- **macOS 11.0 (Big Sur) or later** - Recommended for best compatibility with AeroSpace
- **Apple Silicon or Intel** - Both architectures are supported

### Hardware Requirements

- **RAM**: Minimum 4GB (8GB recommended for smooth LazyVim experience)
- **Disk Space**: ~500MB for all tools and configurations
- **Display**: Any resolution (Retina display recommended for best font rendering)

## Required Tools

These tools must be installed for the configuration to work:

### Core Applications

| Tool | Minimum Version | Recommended Version | Purpose |
|------|----------------|---------------------|----------|
| **Neovim** | 0.9.0 | 0.10.0+ | Text editor and IDE |
| **Tmux** | 3.0 | 3.3+ | Terminal multiplexer |
| **Alacritty** | 0.12.0 | 0.13.0+ | GPU-accelerated terminal emulator |
| **AeroSpace** | 0.1.0 | Latest | Tiling window manager for macOS |
| **Homebrew** | 3.0.0 | Latest | Package manager for macOS |

### Required Dependencies

| Dependency | Installation | Purpose |
|------------|--------------|----------|
| **Git** | `brew install git` | Version control and cloning configs |
| **Curl** | Pre-installed on macOS | Downloading resources |
| **Xcode Command Line Tools** | `xcode-select --install` | Compiling certain packages |

### Fonts

<Warning>
  The 0xProto Nerd Font Mono is essential for proper icon rendering in Neovim and the terminal. Without it, you'll see broken characters.
</Warning>

| Font | Installation | Used In |
|------|--------------|----------|
| **0xProto Nerd Font Mono** | `brew install --cask font-0xproto-nerd-font` | Alacritty, Neovim, Tmux |

**Font Features:**
- Monospace font optimized for programming
- Includes Nerd Font icons for file types, git status, etc.
- Clean and readable at 16pt size
- Excellent ligature support

### Tmux Plugin Manager

| Tool | Installation | Purpose |
|------|--------------|----------|
| **TPM** | `git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm` | Manages Tmux plugins |

## Optional Dependencies

These enhance the experience but aren't strictly required:

### Language Servers (for Neovim LSP)

LazyVim will prompt you to install language servers as needed. Some common ones:

```bash
# Install via Mason inside Neovim (recommended)
# Or install manually:
brew install lua-language-server      # Lua
brew install typescript-language-server # TypeScript/JavaScript
brew install pyright                   # Python
brew install rust-analyzer             # Rust
```

### Development Tools

| Tool | Installation | Purpose |
|------|--------------|----------|
| **ripgrep** | `brew install ripgrep` | Fast file searching in Neovim |
| **fd** | `brew install fd` | Fast file finding in Neovim |
| **lazygit** | `brew install lazygit` | Git UI integration in Neovim |
| **fzf** | `brew install fzf` | Fuzzy finding |

### Additional Fonts

While 0xProto is configured by default, you can install alternatives:

```bash
brew install --cask font-jetbrains-mono-nerd-font
brew install --cask font-fira-code-nerd-font
brew install --cask font-hack-nerd-font
```

## Plugin-Specific Requirements

### Neovim Plugins

The LazyVim configuration includes several plugins with specific requirements:

#### Obsidian Plugin
- **Obsidian vault** - If you want to use the Obsidian integration
- Configured in `lua/plugins/obsidian.lua`

#### LeetCode Plugin  
- **LeetCode account** - For syncing problems
- Configured in `lua/plugins/leetcode.lua`

#### Mason
- **Network connection** - To download LSP servers, formatters, linters
- **curl/wget** - Usually pre-installed on macOS

### Tmux Plugins

Configured in `tmux.conf`:

| Plugin | Purpose | Auto-installed |
|--------|---------|----------------|
| **TPM** | Plugin manager | Manual setup required |
| **catppuccin-tmux** | Theme | Auto-installed via TPM |

### Alacritty Themes

The configuration imports CoolNight theme:
- Theme file: `~/.config/alacritty/themes/themes/coolnight.toml`
- Included in the dotfiles repository
- 200+ other themes available in the themes directory

## Permission Requirements

### AeroSpace

<Warning>
  AeroSpace requires accessibility permissions to manage windows. You'll be prompted on first launch.
</Warning>

**Required Permissions:**
- **Accessibility** - System Preferences > Privacy & Security > Accessibility
- Add AeroSpace to the list and enable the checkbox

### Tmux

No special permissions required.

### Alacritty

No special permissions required, but you may want to:
- Set as default terminal in System Preferences
- Allow in Firewall if enabled

### Neovim

No special permissions required, but some plugins may request:
- **Network access** - For downloading language servers
- **File system access** - For workspace operations

## Version Compatibility

### Neovim LazyVim

- Requires Neovim 0.9.0+
- Lua 5.1+ (built into Neovim)
- Configuration uses lazy.nvim plugin manager
- Compatible with latest LazyVim starter template

### Terminal Color Support

- **True color (24-bit)** - Required for proper theme rendering
- Alacritty: Native support
- Tmux: Configured with `set -g default-terminal "tmux-256color"`
- Terminal test: `curl -s https://gist.githubusercontent.com/lifepillar/09a44b8cf0f9397465614e622979107f/raw/24-bit-color.sh | bash`

## Pre-Installation Checklist

Before proceeding with installation, verify:

- [ ] Running macOS 10.15 or later
- [ ] Homebrew is installed
- [ ] Xcode Command Line Tools installed
- [ ] At least 500MB free disk space
- [ ] Admin access to install fonts and grant permissions
- [ ] Existing configs backed up (if any)
- [ ] Terminal supports true color

## Minimum vs Recommended Setup

### Minimum Setup
```bash
brew install neovim tmux alacritty
brew install --cask nikitabobko/tap/aerospace font-0xproto-nerd-font
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

### Recommended Setup
```bash
# Core tools
brew install neovim tmux alacritty git
brew install --cask nikitabobko/tap/aerospace font-0xproto-nerd-font

# Enhancement tools
brew install ripgrep fd lazygit fzf

# Plugin manager
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```
