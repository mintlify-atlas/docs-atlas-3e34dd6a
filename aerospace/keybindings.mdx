## Keybinding Overview

All Aerospace keybindings use the `Alt` (Option) key as the primary modifier, with `Shift` for secondary actions. This configuration follows Vim-style navigation principles with `h`, `j`, `k`, `l` for directional movement.

<Note>
  On macOS, the `Alt` key is also known as the `Option` (⌥) key.
</Note>

## Layout Management

Control how windows are arranged on your screen.

| Keybinding | Command | Description |
|------------|---------|-------------|
| `Alt + /` | `layout tiles horizontal vertical` | Toggle between horizontal and vertical tiles layout |
| `Alt + ,` | `layout accordion horizontal vertical` | Toggle between horizontal and vertical accordion layout |

<Accordion title="Understanding Layouts">
  **Tiles Layout**: Windows are arranged in a clean grid without gaps, maximizing screen space.
  
  **Accordion Layout**: Windows are arranged with padding (30px by default) between them for better visual separation.
  
  Each layout can be either horizontal or vertical, and pressing the keybinding cycles through orientations.
</Accordion>

## Window Navigation

Move focus between windows using Vim-style directional keys.

| Keybinding | Command | Description |
|------------|---------|-------------|
| `Alt + h` | `focus left` | Focus the window to the left |
| `Alt + j` | `focus down` | Focus the window below |
| `Alt + k` | `focus up` | Focus the window above |
| `Alt + l` | `focus right` | Focus the window to the right |

<Note>
  These keybindings mirror Vim's navigation keys: `h` (left), `j` (down), `k` (up), `l` (right).
</Note>

## Window Movement

Move windows to different positions in the layout.

| Keybinding | Command | Description |
|------------|---------|-------------|
| `Alt + Shift + h` | `move left` | Move current window to the left |
| `Alt + Shift + j` | `move down` | Move current window down |
| `Alt + Shift + k` | `move up` | Move current window up |
| `Alt + Shift + l` | `move right` | Move current window to the right |

## Window Resizing

Adjust window sizes dynamically.

| Keybinding | Command | Description |
|------------|---------|-------------|
| `Alt + Shift + -` | `resize smart -50` | Decrease window size by 50px |
| `Alt + Shift + =` | `resize smart +50` | Increase window size by 50px |

<Accordion title="Smart Resizing">
  The `resize smart` command automatically determines the best direction to resize based on the current layout orientation:
  - In horizontal layouts, it adjusts width
  - In vertical layouts, it adjusts height
</Accordion>

## Service Mode

Enter service mode with `Alt + Shift + ;` for advanced window management operations.

<Warning>
  Service mode contains powerful commands. Always ensure you understand what a command does before using it.
</Warning>

### Service Mode Keybindings

| Keybinding | Command | Description |
|------------|---------|-------------|
| `Esc` | `reload-config`, `mode main` | Reload Aerospace configuration and return to main mode |
| `r` | `flatten-workspace-tree`, `mode main` | Reset layout by flattening workspace tree |
| `f` | `layout floating tiling`, `mode main` | Toggle between floating and tiling layout |
| `Backspace` | `close-all-windows-but-current`, `mode main` | Close all windows except the current one |

### Service Mode - Window Joining

Join windows together in the same container.

| Keybinding | Command | Description |
|------------|---------|-------------|
| `Alt + Shift + h` | `join-with left`, `mode main` | Join current window with window to the left |
| `Alt + Shift + j` | `join-with down`, `mode main` | Join current window with window below |
| `Alt + Shift + k` | `join-with up`, `mode main` | Join current window with window above |
| `Alt + Shift + l` | `join-with right`, `mode main` | Join current window with window to the right |

<Accordion title="Understanding Window Joining">
  Window joining combines two windows into a shared container, allowing them to be managed as a unit. This is useful for:
  - Grouping related windows
  - Creating custom split layouts
  - Managing window relationships
  
  After joining, both windows will move together when you rearrange the layout.
</Accordion>

## Workflow Examples

### Example 1: Organizing a Development Workspace

```bash
# Switch to workspace 1 for coding
Alt + 1

# Open your editor and terminal
# Focus between them
Alt + h    # Focus left (editor)
Alt + l    # Focus right (terminal)

# Resize terminal to be smaller
Alt + Shift + -    # Make it narrower
```

### Example 2: Moving Windows Between Workspaces

```bash
# You're on workspace 1 with a browser window
# Move it to workspace 2
Alt + Shift + 2

# Follow it to workspace 2
Alt + 2

# Go back to workspace 1
Alt + Tab
```

### Example 3: Cleaning Up a Messy Workspace

```bash
# Enter service mode
Alt + Shift + ;

# Reset the layout
r

# This flattens the workspace tree and returns to main mode
```

### Example 4: Working with Multiple Monitors

```bash
# You have workspace 1 on monitor A
# Move the entire workspace to monitor B
Alt + Shift + Tab

# The workspace and all its windows move to the next monitor
```

## Customizing Keybindings

### Modifying Existing Bindings

Edit `~/.aerospace.toml` to change keybindings:

```toml
[mode.main.binding]
# Change focus keybindings to arrow keys
alt-left = 'focus left'
alt-down = 'focus down'
alt-up = 'focus up'
alt-right = 'focus right'
```

### Adding New Bindings

```toml
[mode.main.binding]
# Add a terminal launcher
alt-enter = 'exec-and-forget open -n /System/Applications/Utilities/Terminal.app'

# Add a browser launcher
alt-shift-b = 'exec-and-forget open -a "Google Chrome"
```

### Removing Unused Workspaces

```toml
[mode.main.binding]
# Only keep numbered workspaces and a few letters
alt-1 = 'workspace 1'
alt-2 = 'workspace 2'
alt-3 = 'workspace 3'
alt-b = 'workspace B'  # Browser
alt-m = 'workspace M'  # Mail
alt-s = 'workspace S'  # Slack

# Remove all other workspace bindings
```

## Available Modifiers

All keybindings support these modifiers:

- `cmd` - Command (⌘) key
- `alt` - Option (⌥) key
- `ctrl` - Control key
- `shift` - Shift key

Modifiers can be combined:

```toml
# Example combinations
cmd-alt-shift-f = 'some-command'
ctrl-shift-h = 'some-command'
```

- `keypadClear`, `keypadDecimalMark`, `keypadDivide`
- `keypadEnter`, `keypadEqual`, `keypadMinus`
- `keypadMultiply`, `keypadPlus`

### Arrows
- `left`, `down`, `up`, `right`

## Tips for Learning Keybindings

<Accordion title="Memory Aids">
  **Vim navigation**: `h` (left), `j` (down), `k` (up), `l` (right)
  
  **Pattern**: Basic action = `Alt + key`, Advanced action = `Alt + Shift + key`
  
  **Numbers for workspaces**: `Alt + number` to switch, `Alt + Shift + number` to move window
  
  **Service mode**: Think of `;` (semicolon) as a "command mode" trigger, similar to Vim
</Accordion>
