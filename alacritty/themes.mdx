## Theme System

Alacritty supports a powerful theme system that allows you to customize the color scheme of your terminal. Themes are stored as separate TOML files and imported into your main configuration.

<Note>
  This configuration includes **151 themes** from the official Alacritty themes repository, providing extensive options for customization.
</Note>

## How to Switch Themes

To switch themes, modify the `general.import` line in your `alacritty.toml`:

```toml
general.import = [
    "~/.config/alacritty/themes/themes/THEME_NAME.toml"
]
```

Replace `THEME_NAME` with any theme from the list below. Alacritty will automatically reload the configuration.

## Creating Custom Themes

You can create your own theme by creating a new TOML file in the themes directory with color definitions:

```toml custom_theme.toml
[colors.primary]
background = "#1a1b26"
foreground = "#c0caf5"

[colors.normal]
black   = "#15161e"
red     = "#f7768e"
green   = "#9ece6a"
yellow  = "#e0af68"
blue    = "#7aa2f7"
magenta = "#bb9af7"
cyan    = "#7dcfff"
white   = "#a9b1d6"

[colors.bright]
black   = "#414868"
red     = "#f7768e"
green   = "#9ece6a"
yellow  = "#e0af68"
blue    = "#7aa2f7"
magenta = "#bb9af7"
cyan    = "#7dcfff"
white   = "#c0caf5"
```

Then import it the same way as any other theme.
